Clases principales de Bootstrap

A continuacion les presento algunas de las clases para utilizar en Bootstrap

class = container

Abarca las 12 columnas del grid que nos proporciona CSS, pero deja un margen a los costados

class= container-fluid

Abarca el tamanio del viewport.
